Change inspection lengths

You can change:

  • Marshalling time: the time it takes to marshall the vehicle into the bay.
  • Set up time: the time it takes to set up the bay for this animal.
  • Inspection time: the time it takes for these animals to be inspected.
  • Clean up time: the time it takes for this animal to be cleaned up after.

Making changes to these parameters will effect slot lengths for bookings yet to be created. It is not possible to change the parameters for bookings which have already been created. However, you can edit individual inspection lengths on the inspection details page. Which will then notify the person who made the booking.
